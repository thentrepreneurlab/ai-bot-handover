# Enterpreneurlab Agent

Clone the repo

## Backend

### Techstack

- Python
- Django
- Langgraph
- Langchain
- Uvicorn

### Project Setup

1. Install Python **3.12.10** in system.
2. Change to "dev/backend" to access backend code.
3. Create virtual environment: `python -m venv venv/backend`.
4. Install th requirements: `python -m pip install --upgrade pip && pip install -r requirements.txt`.
5. Move to *src* directory.
6. Run the following command to start the server: `python run_uvicorn.py`


## Frontend

### Techstack

- React
- Tailwindcss

### Project Setup

1. Install node version **v22.20.0**.
2. Change to "dev/frontend* to access frontend code.
3. Install the packages: `npm i`.
4. Run the server: `npm run dev`.
